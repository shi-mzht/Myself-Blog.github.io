main()函数1、安装git

通过Git安装包进行安装，点击安装包一直点击next，傻瓜式操作

2、Git的环境变量的配置

需要绑定用户名和邮箱

###配置用户名

  1. git config –global user.name “username”

//（ “username” 自己的用户名）

###绑定邮箱

  1. git config –global user.email “username@email.com“

/ /(“username@email.com“注测账号时用到的邮箱 )

以上命令执行完毕后，用git config –global –list 命令查看是否成功

3.继续在命令框输入以下命令： ssh-keygen -t rsa

生成ssh,输入以下命令，连敲3次回车键，结束后，去自己的目录下,查看是否有.ssh 件夹

4.将ssh文件夹中的公钥(id_rsa.pub)添加到Gitee管理平台中

5.查看自己的公钥是什么：cat /.ssh/id_rsa.pub

查看配置是否成功，第一次需要输入：yes

ssh -T git@gitee.com

[注意：git remote rm origin如果自己的用户名配置错误，则可以使用此代码更改用户名]
